# DevOps War Room Standards
- Environment API: Gymnasium-style (step, reset, state).
- Core Mechanism: `tick()` function must execute every step to simulate time passing.
- State Schema: Must be a Python dictionary tracking 'services', 'metrics', and 'alerts'.
- Multi-Role: Support SRE, Dev, and Manager roles with different observation filters.
- Output Preference: Generate unified diffs for existing files; full files only for new ones.
